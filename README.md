# Sistema de Gerenciamento de Clientes (CRUD)

Projeto desenvolvido com o objetivo de praticar e aplicar conceitos de desenvolvimento backend utilizando Java e Spring Boot.

---

##  Sobre o projeto

Este sistema realiza operações básicas de um CRUD (Create, Read, Update e Delete) para gerenciamento de clientes.

A aplicação segue o padrão de arquitetura MVC, separando responsabilidades entre camadas como Controller, Service e Repository.

---

## Tecnologias utilizadas

* Java
* Spring Boot
* Maven
* Thymeleaf (para renderização de páginas)
* HTML / CSS

---

## Funcionalidades

*  Cadastro de clientes
*  Listagem de clientes
*  Atualização de dados
*  Remoção de clientes
*  Tratamento de exceções personalizado

---

## Estrutura do projeto

O projeto está organizado em camadas:

* **Controller** → Responsável pelas requisições
* **Service** → Regras de negócio
* **Repository** → Acesso aos dados
* **Model** → Representação das entidades
* **Exception** → Tratamento de erros

---

## Como executar o projeto

### Pré-requisitos:

* Java 17+
* Maven

### Passos:

```bash
# Clonar o repositório
git clone https://github.com/B1tar4es/crud-java.git

# Entrar na pasta
cd crud-java

# Executar o projeto
./mvnw spring-boot:run
```

Depois disso, acesse no navegador:

```
http://localhost:8080
```
